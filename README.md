📼 Blockbuster Entertainment Hub Simulator 
-------
A full-stack-inspired digital rental simulator that brings back the 90s video store experience. This project uses the TMDB API to create a functional "Rent & Return" ecosystem for movies, series, cartoons, and games.

🎮 Features
-----
The Rental Experience: A fully functional "Rent" and "Return" system. Titles are added to your personal "Current Rentals" shelf and persisted across sessions.

Unified Search: Search across the entire TMDB database to find any title in the catalog.

Dedicated Hubs: Specialized pages for:

Movies: The latest Hollywood hits.

TV Series: Binge-worthy shows.

Cartoons: Animated classics for all ages.

Games: Integration for gaming titles (via external API).

Custom Top 5: Every category page features a "Top 5" ranking system where users can curate and save their favorite titles.

Global Translation: A toggle button to switch the entire UI and API data between English and Portuguese (PT-BR).

Persistent Storage: All rentals, Top 5 lists, and language settings are saved to localStorage.

🛠️ Tech Stack
-----
HTML5: Modular structure for multi-page navigation.

CSS3: Retro "Blockbuster" branding (Yellow/Blue theme) using CSS Variables and Grid.

JavaScript (ES6+): - Fetch API: Asynchronous calls to TMDB.

Data Persistence: LocalStorage management for the "Rental" logic.

State Management: Handling language toggles and ranking updates.
